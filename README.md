# MVP Voteroads Frontend

Esse repositório contém o frontend de um MVP do curso de pós-graduação em Engenharia de Software da PUC-RIO.

A ideia é ter um front end simples, com um único arquivo html, usando apenas css e javascript, sem frameworks, comunicando com uma api Flask.

O produto proposto é uma página onde é possível publicar o roadmap de um produto e as features podem ser votadas pela comunidade, para servir de insumo na priorização do desenvolvimento.
